# Web Project for Elderly Assistance

## Overview
This web project is designed to provide a user-friendly platform that connects elderly individuals with assistance services. The main goal is to create a supportive environment where users can easily access resources and information tailored to their needs.

## Features
- A welcoming main menu page with a clear description of the website's purpose.
- A prominent "Start Now" button that guides users to begin their journey.
- A visually appealing collage of images showcasing elderly individuals receiving assistance, emphasizing the community and support available.

## Setup Instructions
1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Install the necessary dependencies using npm:
   ```
   npm install
   ```
4. Open `src/index.html` in your web browser to view the project.
